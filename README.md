# LCL-Level Challenge List

Geometry Dash community level challenge leaderboard with admin controls.

## Installation

```bash
npm install
```

## Run

**Development:**
- Terminal 1: `npm start` (Frontend)
- Terminal 2: `npm run server` (Backend)

**Production:**
```bash
npm run build
npm run server-prod
```
